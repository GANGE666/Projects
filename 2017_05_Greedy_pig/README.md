# Greedy pig ———— C语言课大作业项目

### 基本介绍
此项目是2017年大一下学期开设的高级程序设计语言课的大作业项目，我该门课程成绩97，绩点4.0。

这是一个基于Unity，使用C#制作的一个小游戏，其中包括使用3ds max调整、导入模型，使用VS编写脚本，在Unity中调整相关参数等，具体介绍在[开发设计文档](./开发设计文档.md)中有介绍。
