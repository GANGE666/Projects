# 数字时钟 ———— 数字逻辑实验课大作业项目

### 基本介绍
此项目是2017年大二上学期开设的数字逻辑实验课的大作业项目，我该门课程成绩99，绩点4.0。

此项目通过protel设计电路，并设计PCB，老师交付给厂商生产后，焊接芯片，实现数字时钟，具体介绍在[数字逻辑实验报告](./数字逻辑实验报告.md)中有介绍。
