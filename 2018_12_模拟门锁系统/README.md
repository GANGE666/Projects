# 模拟门锁系统 ———— 嵌入式系统安全实验课大作业项目

### 基本介绍
此项目是2018年大三上学期提前选修的嵌入式系统安全实验课程设计，我该门课程成绩95，绩点4.0。

此项目一个基于arm s2410实验台，基于linux操作系统，实现了输入正确密码，电机启动，并可以修改密码。且为了稳定、安全的要求，对于诸多细节进行了处理，并对于侧信道攻击、芯片逆向攻击有一定的防御能力，具体介绍在[嵌入式系统安全实验报告](./嵌入式系统安全实验报告.md)中。
